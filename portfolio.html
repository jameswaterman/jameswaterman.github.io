In 2018 I had just finished college and found myself rather lost with what to do next. Whilst away travelling I decided I wanted a website where I could share my photography. With a website being the goal I decided I would learn to make one myself as time is something I had plenty of.

After learning the basics from various websites, I decided to use platforms such as Codecademy to further my knowledge and provide myself with a solid foundation to further progress on my own. The more I worked on projects, the more my eyes were opened to what could be achieved through coding; sparking my enthusiasm to continue improving my work.

Since then I have worked with various languages and frameworks, such as HTML5, CSS3, JavaScript, Sass, React.js, Express.js, Node.js and SQL.

Projects

Ingredients Calculator

Ingredients Calculator is a web app I created that allows you to take the ingredients from a recipe and then increase/decrease them based on how many people you're cooking for. I produced the app using JavaScript and CSS with the React framework to produce a fast and lightweight app that doesn't require the use of a server or database. GitHub

Jamming

Jamming is a web app I built that interacts with the Spotify API, allowing you to create and save new playlists to your Spotify account. I built this web app with the react framework, along with CSS. GitHub

Business Website Template

I built this fully responsive buisness website template using primarily HTML and CSS, as well as JavaScript for the animation of the navigation. Throttling has been used to control how often functions attached to a scroll event are being executed to help improve performance. GitHub

Ravenous

Ravenous is a web app I built which interacts with the Yelp API to provide you with quick reviews of restaurants in a given area. Ravenous allows you to sort your search results based on best match, highest rating or most reviews. I built this app using React alongside CSS. GitHub

Colmar Academy

Colmar Academy is a fully responsive website template for "Colmar Academy". It has been built completely with HTML and CSS. GitHub
